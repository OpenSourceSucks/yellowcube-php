# YellowCube PHP Documentation

## Installation

```
sudo easy_install pip
sudo pip install dexy
sudo pip install markdown
```

or

```
pip install .
```

## Building

 * `cd docs`
 * Build files `dexy`
 * Start a server to deliver the docs `dexy serve`
